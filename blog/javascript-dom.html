What is JavaScript and its relationship to HTML and CSS.

JavaScript could be referenced as the brain of a webpage. In a simple term, it assists with page movement, response, and all other actions that need to be taken.


  • HTML shows what's on the page.
  • CSS makes it pretty.
  • JavaScript makes it dynamic.


What is Control Flow?

Control Flow is the order of the structure of statements that needs to be followed. It's like a routine of whatever you are doing.

For example, if you have a routine after waking up, everything would be fine if followed. However, if you are off a little from your routine, usually something goes wrong.



What is Loops?

Loops are a sequence of instructions that is continually repeated until a specific condition is reached.

For example, if you were to run 2 kilometers on a running track that is 400m long in an oval shape, a sequence of instructions could be referred to the action of running and a certain condition would be how long you need to run, in this case 2 kilometers.



What is DOM?

DOM is an abbreviation of Document Object Model. It's there to help interact with all the HTML elements.

DOM is a copy of things inside the HTML document represented in an object type.

This way, it is easier to access the HTML elements when dealing with JaveScript. Browsers provide a 'document' JavaScript object with a large number of properties to interact in all sorts of ways.

How to access HTML elements using the DOM.

You can access HTML elements through the object 'document'. Just like how you normally access an object type, you would follow the structure 'document.___' way.

The three most common things to query by are:

  • by tag name (element name, such as 'p' or 'div')
  • by class name (class attribute e.g. class="container")
  • by id (id attribute e.g. id="logo")

The three different methods are used in different situations. But, ideally, to access individual elements in the HTML document, id is the most common way to go.

On the other hand, if you want to access multiple elements at once, you would access them with a tag or class name.

The way you can access these is through 'document.querySelector("__")', and you receive the information related to the query you selected in the object type.



How to use HTML elements using the DOM.

You can manipulate the elements in the same way you use an object type variable. We can use '.innerHTML' attribute to both get and set the inner HTML tags.

For example:

  • 'let value = document.querySelector('__').innerHTML' will set the variable 'value' to the inner HTML value of whichever query is selected.
  • 'document.querySelector('__').innerHTML = "I have changed the value!"' will change the inner HTML value of whichever query that is selected to 'I have changed the value!'.

We need to note that selecting a class or the tag, will only access the first element of the HTML document. In order to select all the queries, you would use '.querySelectorAll' instead.



More examples of things you can do with DOM.

There are a few more commonly used attributes through the 'querySelector' object like '.setAttribute' and '.addEventListener'.


.setAttribute()

This helps us modify a few things like the style of the element, src, or href.

For example:

  • 'exampleSpan.setAttribute('style', 'color: red; font-weight: bold')' modifies the 'exampleSpan' with the style specified in the 'setAttribute' bracket.

.addEventListener()

There are many ways to use the 'addEventListener'. This is the main way to interact with the click, keystrokes, mouse moves, and more.

The 'addEventListener' has the attribute 'event' to specifically achieve what you want to do.

For example:

  • document.querySelector('#test-button').addEventListener('click', (event) => { console.log(event.target) }) will follow the id query '#test-button' which can receive a button click, and once the action happens, it will have the accessibility to the inner HTML value of the button.


What is an Array and objects?

Array is like a tube container that you can store items into. Object is almost like an array but it can store additional details for each item.

  • Array - It is used with square brackets '[]' and there are many methods you could interact with arrays like '.length', '.sort()', '.forEach()', and many more.
  • Objects - It is used with curly brackets '{}' and it contains keys and values. For example, 'obj = {key1 : value1, key2 : value2}' and to access, 'obj.key1' or 'obj[key1]' would give us 'val1'.


What are functions?

Functions are like recipes of foods. It is like a page of instructions which can be reused. It is helpful when your codes are repetitively used, you could simply premake a function and just use the function name to execute the instructions.
