Position Property

The position property specifically allows you to deal with the position of the element in different ways. It can be adjusted by the top, bottom, left, and right properties.


There are 5 different values used with position property:


  • static
  • relative
  • absolute
  • fixed
  • sticky

Each value has unique behaviors.


position: static;

By default, HTML elements are positioned as static.

Elements with this value are not affected by any of the adjusting properties (top, bottom, left, and right).

Instead, they are positioned naturally, following the rest of the structure in HTML.

position: relative;

Relative position can be adjusted with the adjusting properties.

Just like static value, it follows the flow of the HTML structure at the start, but can be re-positioned.

Even if it was re-positioned, other contents treat it as a static position ignoring the adjustment.


Example:

游릳 relative - top: 0px, left: 0px

游릳 relative - top: 200px, left: 200px

position: absolute;

Absolute position can also be adjustable just like relative position.

However, the absolute position is affected by its parent element as well.

If an absolute position element has a parent element with a relative position, when the relative position element gets adjusted, the absolute position element also gets adjusted with the same amount.

So, the parent adjustment is added to its original adjustment.


Example:

游릴 absolute - top: 0px, left: 0px

游릳 relative - top: 200px, left: 200px

游릴 absolute - top: 100px, left: 100px

position: fixed;

Fixed position is a little special. It has a direct relationship with the window.

It is always fixed in the position you have set. It does not get affected by any elements or scrolling.


Example:

fixed green box in the corner - scroll up

fixed green box in the corner - scroll down

position: sticky;

Sticky position is a mixture of relative and fixed position.

Just like fixed position, the sticky position does not leave the screen of a window. If you try scrolling too far, it get fixed in the position you have set.


Example:

sticky grey nav side bar - scroll up

sticky grey nav side bar - scroll down
